Ryaz Animation

Published on August 16, 2021 by Khushboo Dhiman on Web Technologies

Exciting Task Day.

Tasks Involved

1. Designing the desired outcome.

2. Learning about Anime.js library.

3. Deciding the flow.

4. Started with the logo.


1. Designing the desired outcome.

Again starting with deciding the desired outcome of task I started my animation task. I love Figma. So again I chose figma for creating this and above I've included the image of design.

2. Learning about Anime.js library.

To create animation using JavaScript, I needed the help of some library to make my task easy so I started exploring the available JavaScript libraries. And eventually came to Anime.js and found this one light weight, easy to use and quick for a start.

I found some good resources for learning this which I'll be including at the end.

3. Deciding the Flow.

So, after designing I create the simple flow of animation in my mind. How I want things to show up and hide?

4. Started with the logo.

Today, after all this stuff, I was able to just create animation for the R logo. First, I tried to take them in from different sides and then combine them but later I changed my mind and went for scaling stuff.

Anime.js Documentation: https://animejs.com/documentation/
Blog Post for Anime.js: https://www.section.io/engineering-education/user-interface-animation-with-animejs/

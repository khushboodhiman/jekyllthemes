React Learning (Day-4)

Published on August 10, 2021 by Khushboo Dhiman on Web Technologies

Another Learning Day.

Whole day revolved around looking for solution to implement the cart thingy in my shopping cart project. Learnt a lot things but still didn't came to a solution. No worries! Will try again tomorrow.

Key things that I learnt today.

1. Checking the Shopping app module in react course.

2. JSX Limitations.

3. React Portals.

4. React with Redux.


1. Checking the Shopping app module in react course.

The React course I was refering to had a module for creating some Food ordering app, So I thought I can find the solution to implement cart there. But there he was using a portal to create the cart which he had explained in some previous module which I skipped. So I changed my mind and went to that module for JSX Limitations and Portals.

2. JSX Limitations.

First is that we need to always wrap the component in something when we want to return it, which is obvious because there we use JavaScript return which doesn't support multiple elements to return. We can wrap elements around divs or [] create array, or just empty tags. But this is not the ideal approach.

In React, we have a component that just returns the children i.e. the content or elements in its opening and closing tags. Because that component returns props.children.

We can now easily wrap our elements around this Fragment provided by React and get a relief from the unnecessary divs in our dom.

3. React Portals.

Creating components like modals, pop ups in a div and giving them styling like a modal and addign some event listeners to make them look like an actual modal isn't the ideal way. So, I came to learn about React Portals.

In index.html create two another divs along with root. One for backdrop and another for modal. To use createPortal function of Portals we need to import ReactDOM. createPortal takes two arguments - what-to-render(JSX code) and where-to-render.

4. React with Redux.

By exploring google, came to a website which teachs to create cart using React and redux too. Redux because it is mostly used to handle states of our website and has easy implementation for e-commerce things. Learnt a bit about how to import redux and create a store and use Providers. Rest of this will continue tomorrow. Good Night!

React and Redux link: https://medium.com/@ayabellazreg/make-a-simple-shopping-cart-app-using-react-redux-1-3-fefde93e80c7

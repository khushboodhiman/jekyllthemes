CSS Grid System

Published on July 23, 2021 by Khushboo Dhiman on Web Technologies

Introduction

CSS Grid is a 2 dimentional grid-based layout system. In which, we consider rows and columns and make the layout in those rows and columns. So we dont need to use floats, positioning, and all out-dated methods to create our website's layout. Flexbox and grid are the new libraries to easily create our website layouts.

Terminology for Grid System

For properly understanding the CSS grid system, we need to understand the terminology, so when it comes to applying grid system to your website we dont get confused or find it difficult.

Grid Container

The parent element in which whole grid system is implemented is called the grid container. It is the direct parent of grid elements. By applying the display: grid property, we can implement grid in its direct children.

Grid Item

The direct children of the parent div or the grid container is called the grid item or items.

Grid Line

The line dividing the rows and columns is called grid line. There are column grid lines and row grid lines.

Grid Cell

The item that is consituted by combining row and column is called a grid cell. We can expand an item to a certain number of grid cells.

Grid Area

The area which is consituted by combining certain cells is called grid area.

Grid Container Properties

Display

Defines the parent element as the grid container. There are two possible options: inline-grid, grid

Grid-template-rows

Defines the template for rows in the grid. We can provide a number in pixels here / % / fractions (fr) / auto.

Grid-template-columns

Defines the template for columns in the grid. We can provide a number in pixels here / % / fractions (fr) / auto.

Grid-template-areas

Defines the template for rows & columns combined in the grid. We can provide rows in next lines and columns in rows line.

Column-gap

Defines the gap or gutter in the grid columns. Grid is so smart that it doesn't applies this gutter or margin on the starting and ending of the grid container.

Row-gap

Defines the gap or gutter in the grid rows.

Gap

Defines the combined gap for rows and columns.

Justify-items

Defines the justification(alignement in rows) for grid items. Values can be start, end, center or stretch. Same goes for align items property.

Align-items

Defines the vertical alignment of the grid items.

Grid Child Properties

Grid-column-start / Grid-column-end / Grid-row-start / Grid-row-end

Defines the location of grid items. We can define the column start and end, row start and end of a grid item using these properties.

Grid-column / Grid-row

Shorthand of using Grid-column for grid-column-start and grid-column-end. In this you can specify both the values separated with /.

Justify-self

The property to justify the grid items is justify self. Values can be start, end, center and stretch.

Align-self

Defines the vertical alignment of grid items. Values can be start, end, center and stretch.

Random Player Generator

Published on August 12, 2021 by Khushboo Dhiman on Web Technologies

A Small React app.

Tasks Involved

1. Deciding the desired outcome.

2. Generating random names.

3. Adding Skip and Who's next?.

4. Styling the project.


1. Deciding the desired outcome.

Without having an image in mind How can anyone start something? So I just closed my eyes, thought of how I want this to work and then started.

2. Generating random names.

Main task was to generate random names from an array. So I did this as the very first step. Create an array of names and generator random names from that array on the screen.

First I did this on refresh and then I moved this on a button click.

3. Adding Skip and Who's Next?

Next, I added two buttons Skip to skip one's turn and Next to display the next person's turn.

4. Styling the Project.

At the end, when everything is done, I added a very simple styling to the components. And that's it :smile
